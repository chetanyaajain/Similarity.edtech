# User Guide

## Who This Is For

This app is for teachers, instructors, and academic reviewers who want to compare student assignments inside a batch and look for unusually similar work.

## Typical Flow

1. Sign in
2. Create a batch
3. Upload assignments into that batch
4. Open the results page
5. Review similarity scores and matched sections
6. Generate a report

## Important Behavior

- Similarity needs at least 2 assignments in the same batch
- If you upload only 1 file, the system can summarize and extract keywords, but it cannot compare similarity yet
- Clean text files usually produce the best results
- Scanned PDFs are supported through OCR, but quality depends on the scan

## Best Upload Types

Best:
- text-based PDF
- DOCX
- TXT

Weaker:
- blurry scanned PDF
- image-heavy PDF
- handwritten scans

## What the Scores Mean

- Higher similarity percentage means the pair is more alike
- High semantic score can indicate paraphrasing, not just copy-paste
- Originality score is based on how similar a submission is to others in the same batch

## What to Do If Results Look Weak

- Upload cleaner source files if possible
- Prefer text-based PDFs over scanned copies
- Use DOCX or TXT when available
- Make sure multiple submissions are in the same batch
